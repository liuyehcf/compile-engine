# 编译原理

该项目的终极目标是 DIY 一个简单的编译器

## 词法分析

__正则表达式引擎__

* base package: `org.liuyehcf.grammar.rg`
* 包括nfa自动机以及dfa自动机
* 支持以下正则表达式通配符: `.`、`?`、`*`、`+`、`|`
* 支持量词`{}`
* 支持部分转义，包括`\d`、`\D`、`\w`、`\W`、`\s`、`\S`
* 支持`[]`
* nfa自动机支持捕获组（nfa转dfa时，捕获组信息会丢失，因此dfa自动机尚不支持捕获组。目前还没有很好的解决方法）

## 语法分析__
